# FlexUI (work in progress)
FlexUI is a DSL like SwiftUI to make View, currently in early development

# Installation
### Only CocoaPods is supported
in Podfile

```pod 'FlexUI', :git => 'https://github.com/WillPromise/FlexUI.git'```
